# Anatomy
>==Search== enables users to specify a word or a phrase to find relevant pieces of content or filter a long list of elements. Commonly represented as a Text Input with a little magnifying glass inside of it.
![Anatomy for Number Input Component](https://ui-guideline-components.netlify.app/components/number-input/anatomy/desktop/images/number-input.png)
- Possible Names: ==Search==, ==Search Input==, Filtered Search, Global Search, ==Autocomplete==, Search Bar
- Grouping: `Inputs`
